Transaction hash signing (Blind Signing)
Warning

Signing hashes from untrusted sources can lead to loss of funds.
Transaction hash signing (Blind Signing) feature is intended only for advanced users.
You must ALWAYS validate that the hash you are signing matches the expected transaction.
This function requires 'Blind Signing' to be enabled on the Kadena Ledger App.
